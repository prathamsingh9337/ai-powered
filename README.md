# 🩺 AI-Powered Health Assistant

An intelligent AI-powered healthcare assistant built using the MERN stack and modern LLM technologies. The project uses Retrieval-Augmented Generation (RAG) with ChromaDB and OpenRouter API to provide contextual and real-time healthcare responses.

---

## 🌐 Live Demo

https://ai-powered-one.vercel.app

---

## 🚀 Features

- 🤖 AI-powered healthcare chatbot
- 🔍 RAG-based contextual response system
- 🧠 Vector search using ChromaDB
- ⚡ Real-time LLM responses with OpenRouter API
- 🔐 Secure backend APIs
- 📱 Responsive frontend UI
- 🌐 Full-stack deployment

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/health-ai-assistant.git
cd health-ai-assistant
```

### 2. Install Dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGODB_URI=your_mongodb_uri
OPENROUTER_API_KEY=your_openrouter_api_key
```

---

## ▶️ Run the Project

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
npm start
```

---

## 🧠 How RAG Works

1. User sends a healthcare-related query
2. Query is converted into vector embeddings
3. ChromaDB retrieves the most relevant context
4. Context is passed to the LLM
5. AI generates a contextual response
